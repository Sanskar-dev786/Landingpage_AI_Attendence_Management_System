# AI Attendance Landing Page

A modern and responsive landing page for the **AI Attendance System**, designed to showcase the platform's capabilities and provide easy access to the attendance application.

## 🚀 Overview

The AI Attendance System is a smart attendance management solution that helps teachers and students manage attendance efficiently through an intuitive and modern interface.

This landing page serves as the public-facing website for the project and highlights its key features, workflow, and access points.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Sanskar-dev786/ai-attendance-landing.git
cd ai-attendance-landing
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### macOS/Linux

```bash
source venv/bin/activate
```

#### Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

The application will start on:

```text
http://127.0.0.1:5000
```
